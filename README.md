# Smart Support Chatbot

A full-stack chatbot web application built using Python (Flask) and JavaScript.  
The application simulates a customer support assistant for small businesses, answering common queries such as prices, opening hours, bookings, and services.

---

## Overview

This project demonstrates how to build a simple chatbot system using a frontend interface and a backend API.  
Users can interact with the chatbot through a web interface, and the backend processes the request and returns relevant responses.

---

## Features

- Chat-based user interface in the browser
- Backend API built using Flask
- Handles common customer queries (prices, hours, bookings, services)
- Real-time interaction between frontend and backend
- Rule-based chatbot logic using keyword matching

---

## Tech Stack

- Python (Flask)
- Flask-CORS
- HTML
- CSS
- JavaScript
- REST API
- JSON request/response handling

---

## Example Questions

Users can ask questions like:

- What time do you open?
- How much is a haircut?
- Can I book an appointment?
- Where are you located?
- What services do you offer?

---


## How to Run
1. Install dependencies:
pip install flask flask-cors

2. Run server:
python3 app.py

3. Open in browser:
http://127.0.0.1:5000
