# Smart Support Chatbot

A full-stack chatbot web application built using Flask and JavaScript.

## Features
- Responds to customer queries (prices, hours, bookings)
- REST API using Flask
- Frontend chat interface
- Real-time interaction

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- JSON API

## How to Run
1. Install dependencies:
pip install flask flask-cors

2. Run server:
python app.py

3. Open in browser:
http://127.0.0.1:5000
